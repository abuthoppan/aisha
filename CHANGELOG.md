## [1.0.0] — 2026-03-10

### Added
- Full homepage with hero, marquee, about, pillars, framework, and blog sections
- Blog archive with category filtering
- Individual post layout with author bio, tags, and post navigation
- Framework page with full LSF five-phase breakdown
- About page
- Mailchimp newsletter subscribe (JSONP, no backend)
- Mobile-responsive navigation with hamburger menu
- Scroll reveal animations
- Two sample posts
- Data-driven pillars and phases via `_data/` YAML files
- `README.md`, `CONTRIBUTING.md`, `LICENSE`, `LICENSE-CONTENT`
- GitHub Actions deployment workflow
- Issue templates for bug reports and tutorial requests
